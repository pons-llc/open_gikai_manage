市民に開かれた議会情報を、
低コストに運用できる形で。

open_gikai は、地方議会の会議日程・議題・委員会・議員・お知らせ・会議資料を 誰でも認証不要で閲覧できるようにする、地方自治体向けのオープンソースシステムです。 Cloudflare Workers + D1 + R2 のみで構成し、従量課金の予算スパイクを設計段階から防ぎます。

デモを見る(ダミーデータ) セットアップ手順(README)

特徴

「閲覧系は完全公開・更新系のみ認証」というシンプルな2層モデルと、 Cloudflare の従量課金を前提にした予算スパイク対策を核にしています。

認証不要の公開画面

日程・議題・委員会・議員・お知らせ・資料DLは、市民なら誰でもログインなしで閲覧できます。JS無効でも閲覧系は動作します。

「前の会議終了後」チェーン日程

開始時刻が未定の会議も、「前の会議終了後」という形で自然に表現・表示できます。

2段階の賛否記録

委員会採決→本会議採決という2段階の賛否を、議員ごと・会議ごとにグリッド入力し公開できます。

予約公開(cronなし)

お知らせ・議題の公開予約は読み取り時のフィルタ(published_at <= now)で実現し、cronジョブのコストをかけません。

予算スパイク対策を内蔵

公開GETはゾーンレベルで30分(資料は1日)キャッシュ、R2は総容量クォータでコスト上限を設計段階から確保しています。

デジタル庁デザインシステム準拠

DADS v2.12.0 のトークンを採用し、アクセシブルで自治体らしい見た目を最初から実現します。

技術スタック

追加のホスティング費用や外部認証SaaSに依存せず、Cloudflareの枠内で完結します。
